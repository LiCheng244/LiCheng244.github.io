iOS开发 - UIButton 防止重复点击

发表于 2018-05-05   |   分类于 iOS__02【 UI 知识 】   |     |   阅读次数

引言


I. 禁止交互

使用 UIButton 的 userInteractionEnabled 属性。在点击后，禁止UIButton的交互，直到完成指定任务之后再将其 button 的交互打开即可。


II. 延迟执行

使用 performSelector:withObject:afterDelay:和cancelPreviousPerformRequestsWithTarget

使用这种方式点击 UIButton 的时候，自动取消掉之前的操作，延时1s后执行实际的操作。重复点击时，每点击一次都会取消掉上一次点击要执行的函数，然后1秒后执行实际的函数。这种方式看起来会比第一种和第三种稍微延迟执行实际的操作。


III. Runtime

使用runtime来对sendAction:to:forEvent:方法进行hook. UIControl的sendAction:to:forEvent:方法用于处理事件响应.
如果我们在该方法的实现中，添加针对点击事件的时间间隔相关的处理代码，则能够做到在指定时间间隔中防止重复点击。
